Welcome to AMA Bot! This bot was made to create a running AMA board for people who want to be asked questions.

To get started, you can make your AMA using /addme.

You can ask questions on other people's AMAs using /ama {/users ID} {text}.

You can send a reply to the person who asked you an AMA question by using /reply {question ID} {text}.

To learn more, use /help.
